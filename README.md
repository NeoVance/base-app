base-app 1.1
========

The base application in PhalconPHP
***
Components:
--------
* Bootstrap file
* Config file
* CLI file
* Frontend/Backend modules
* Library
 * Auth
 * Arr
 * Debug
 * Email
 * I18n
 * Image
* User
 * Models
 * Auth schema mysql
* Twitter Bootstrap

Configuration:
--------
1. Set base_uri in /app/common/config/config.ini
2. Use /auth-schema-mysql.sql to create required tables
